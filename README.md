## dotfiles

**Note to self:** To recreate the symlinks you do the following from the Home folder:

    $ rm -r .gitconfig
    $ ln -s .dotfiles/gitconfig .gitconfig

For more, [see the Wiki](https://github.com/stormwarning/dotfiles/wiki).
